# TodoPet 기획 개선 제안서

작성일: 2026-02-23
기준 자료: `README.md`, `PROGRESS.md`, `src/pages/*`, `src/components/pages/*`

## 1) 현재 서비스 강점 요약

- `Todo 완료 -> 보상 -> 펫 성장` 루프가 명확해서 동기부여 구조가 좋습니다.
- 주/월 캘린더, 카테고리, 펫 상태, 랭킹, 마이페이지까지 핵심 화면이 갖춰져 있습니다.
- 랭킹/인벤토리/토스트 등 "게임적 피드백" 요소가 이미 구현되어 확장 여지가 큽니다.

## 3) 상세 제안

### 3-1. 첫 사용 10분 온보딩 개선 (P0)

- 문제
  - 로그인 후 바로 기능이 많아 보이며, "무엇부터 해야 보상을 받는지"가 처음엔 모호할 수 있습니다.
- 제안
  - 첫 로그인 시 3-step 가이드: `카테고리 1개 생성 -> 오늘 할 일 1개 생성 -> 완료 체크`.
  - 완료 즉시 "첫 성장 보상"을 지급해 핵심 루프를 강하게 체험시킵니다.
- 최소 범위(MVP)
  - 온보딩 모달/툴팁 3단계 + 완료 여부 저장 + 최초 보상 1회 지급.
- 기대 효과
  - 활성화율(가입 당일 첫 완료 경험 비율) 증가.

### 3-2. 인벤토리 초과 보상 처리 개선 (P0)

- 문제
  - 현재도 인벤토리 초과 안내는 있으나, 유저 입장에서는 "열심히 했는데 보상을 잃었다"는 손실감이 큽니다.
- 제안
  - 보상을 버리지 않고 `보상 보관함(우편함)`에 임시 보관.
  - 인벤토리 비우면 수령 가능하게 하여 이탈 요인을 줄입니다.
- 최소 범위(MVP)
  - 보관함 탭 1개 + 보관 아이템 만료 정책(예: 7일) + 일괄 수령.
- 기대 효과
  - 완료 행동 대비 보상 체감 개선, 불만/이탈 감소.

### 3-3. 반복 할 일(Recurring Todo) 추가 (P0)

- 문제
  - 일상 루틴(운동, 공부, 물 마시기 등)은 반복 생성이 필요해 현재 입력 피로가 큽니다.
- 제안
  - 생성 시 반복 주기 옵션: `매일/평일/주 n회/매주 특정 요일`.
  - 캘린더 날짜별 자동 생성으로 기록 누락을 줄입니다.
- 최소 범위(MVP)
  - 매일/매주(요일) 2개 패턴부터 시작.
- 기대 효과
  - 주간 완료 수 증가, 앱 재방문율 상승.

### 3-4. 연속 달성(Streak) + 주간 회고 카드 (P0)

- 문제
  - 현재는 단건 완료 보상 중심이라 "장기 습관화" 동기가 약할 수 있습니다.
- 제안
  - `n일 연속 완료` 지표를 추가하고, 7일/14일/30일에 보너스 지급.
  - 마이페이지에 "이번 주 회고 카드(완료율, 가장 많이 미룬 카테고리)"를 표시.
- 최소 범위(MVP)
  - 연속일 계산 로직 + 주간 요약 카드 1종.
- 기대 효과
  - D7/D30 리텐션 개선, 복귀 동기 강화.

### 3-5. 랭킹 2트랙(전체 + 친구) 분리 (P1)

- 문제
  - 전체 랭킹만으로는 상위권 외 유저가 성취감을 느끼기 어렵습니다.
- 제안
  - 전체 랭킹 유지 + 친구 랭킹(소규모 리그) 추가.
  - 친구 리그는 주간 메달/칭호를 별도로 제공.
- 최소 범위(MVP)
  - 친구 20명 제한 리그 + 주간 초기화 유지.
- 기대 효과
  - 랭킹 참여율 증가, 사회적 리텐션 강화.

### 3-6. 목표(카테고리)와 보상 연결 강화 (P1)

- 문제
  - 카테고리는 존재하지만 "목표 달성 감각"이 약할 수 있습니다.
- 제안
  - 카테고리별 주간 목표(예: 운동 5회) 설정.
  - 목표 달성 시 펫 스킨/배지/특수 아이템 지급.
- 최소 범위(MVP)
  - 카테고리별 목표 횟수 + 달성 배지 1종.
- 기대 효과
  - 카테고리 기능 사용률 증가, 동기부여 다층화.

### 3-7. 리마인더/알림 설계 추가 (P1)

- 문제
  - 할 일을 잊어서 미완료되는 경우를 제품이 충분히 방어하지 못합니다.
- 제안
  - 인앱 알림 우선 도입 후 웹 푸시 확장.
  - 알림 타입: `오늘 미완료`, `연속 달성 끊기기 직전`, `보상 보관함 만료 임박`.
- 최소 범위(MVP)
  - 인앱 알림 센터 + 1일 1회 리마인더.
- 기대 효과
  - 미완료율 감소, 재방문 증가.

### 3-8. 시즌 이벤트/한정 미션 운영 (P2)

- 문제
  - 기본 루프가 익숙해지면 콘텐츠 신선도가 떨어질 수 있습니다.
- 제안
  - 월간/분기 이벤트 미션(예: 2주 챌린지)과 한정 아이템 운영.
  - 이벤트 캘린더를 마이페이지에 노출.
- 최소 범위(MVP)
  - 월 1회 이벤트 + 한정 아이템 1종.
- 기대 효과
  - 장기 유저 유지율 및 재활성화율 개선.

## 4) 제안 KPI (측정 지표)

- 활성화: 가입 후 24시간 내 첫 Todo 완료율
- 리텐션: D1, D7, D30 유지율
- 참여도: 주간 평균 완료 Todo 수, 카테고리별 완료율
- 게임화: 연속 달성 7일 이상 비율, 랭킹 참여율
- 품질: 보상 분실/불만 관련 문의 건수

## 5) 6주 실행 로드맵 예시

- 1-2주차
  - 온보딩 3-step
  - 보상 보관함
- 3-4주차
  - 반복 Todo
  - 연속 달성 + 주간 회고
- 5-6주차
  - 친구 랭킹
  - 카테고리 목표 연동

## 6) 확인 필요 사항

- 랭킹 기준 시간대(예: KST/UTC) 및 초기화 기준 고정 여부
- 보상 경제 밸런스(일일 최대 보상량, 히든 보상 확률) 조정 정책
- 푸시 알림 도입 시 개인정보/동의 UX 및 운영 주체
